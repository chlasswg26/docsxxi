RequiredOptionalValueExampleDescriptionResult
pagepage = Number to page (1,2,3,4,5 or more)Required : https://yourdomain.com/latest.php

Optional : https://yourdomain.com/latest.php?page=2		Results returning array of json encode

Use 'code' for request post.php		isPaging = Boolean (true / false)
next = Number to next page (1,2,3,4,5 or more if available)
prev = Number to previous page (1,2,3,4,5 or more if available)
